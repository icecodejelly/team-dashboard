Airtable Configuration

Setup Instructions:

  1. Create a new Airtable base or use an existing one
  2. Create a table called "Projects" with these fields:
    • Name (Single line text)
    • Status (Single select: Planning, In Progress, Review, Complete, On Hold)
    • Priority (Single select: Low, Medium, High)
    • Target Date (Date)
    • Bottleneck (Long text)
    • Next Step (Long text)
    • Assignee (Single line text)
    • Progress (Number, 0-100)
    • Days in Pipeline (Number)
  3. Get your Base ID from the Airtable API documentation
  4. Create a Personal Access Token in your Airtable account settings
